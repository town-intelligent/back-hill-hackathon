關於我們

這次我們正透過公益聯盟舉辦的「2022公益後山數位黑客松」活動，背後的想法正是期待能夠，創造更多全台各地數位人才，能夠進入到鄉鎮來，重新為傳統式微的地方，尋求未來數位創生解方！

非常謝謝公益聯盟支持！讓「小鎮文創」、「小鎮智能」、「小鎮風土」與「東華大學」能夠共同攜手協辦這場活動，我們都非常期待能夠為台灣的小鎮、農村與部落帶來不同以往的數位創生未來想像。

  • 小鎮文創
  • 小鎮智能
  • 小鎮風土

自從天空的院子成立以來，我們修復百年建築成為中部地區第一間文化民宿場域，並且致力於在地文化與產業的融合，開創出更多在地老文化結合新創意的發展，而台灣是一座寶島，寶島坐落許多小鎮，小鎮藏有著許多動人的故事。要如何把每個鄉鎮的光點故事，整合、集結、串聯起來，並且引進更多資源協助建立一個永續的文化創意發展經營模式，是【小鎮文創】成立的宗旨與價值。

我們在竹山鎮上，成立的【小鎮智能】，運用區塊鏈技術，發行竹山數位鎮民計畫，越來越上軌道，運用社區貨幣交易，進行竹山足跡計算，運用資料科學技術進行分析，未來能夠成為【小鎮文創】在竹山小鎮的決策系統。

【小鎮風土】-土地與生活，有土斯有財。純粹美好的土地，孕育豐美的蔬果，不難發現這其中的深意。

    Q&A

    🔔🔔🔔大家好!! 大家開始著手撰寫出您的偏鄉永續計劃書了呢? 是否有一些疑問想詢問呢? 以下幫您整理了這次競賽的問答篇!!讓您可以更了解比賽相關細節哦!!



    🚩參賽報名期間:9/16(五)-10/20(四) 🚩參賽資格: 大專院校/碩班在學或者同等學歷之學生，以組為報名資格，可跨校際。 🔔🔔🔔不限制科系!!不一定要會寫程式哦!!只要您有想法!!歡迎您寫出您的偏鄉永續計畫書 🚩錯過了線上說明會，怎麼辦? 這裡有線上說明會的錄影回顧可以讓您更了解參賽的細節哦!! 第三屆青年公益競賽 後山黑客松線上說明會錄影 https://youtu.be/Wy2FohZWvG8



    🔔🔔🔔第三屆青年公益競賽 問答篇 🚩問題: 不住在東部，是否也可以參加呢?

    回答: 可以的!只要提出和您的家鄉有關的偏鄉議題，與東部有連結的方案，皆可報名參加，例如:東部和您的家鄉都有人口外流的問題。



    🚩問題:報名資格只限定是大學跟碩班嗎?

    回答:是的!!本次青年公益競賽以大學跟碩士班學生為主要參加對象。



    🚩問題:不會寫程式，是否也可以參加呢?

    回答:可以的!只要提出您的偏鄉永續計畫書，不管您會不會寫程式，都可以參加本次青年公益競賽，我們將協助程式撰寫部分。



    🚩問題:競賽所提交的計劃，如初選通過參加工作坊，是否還能調整計畫執行項目內容，於demo day當日進行簡報說明?

    回答:可以的，活動承辦單位將再提醒初選入選的12組團隊於指定日期前調整及提交demo day決賽簡報。



    🚩問題:主辦單位將何時發送獎金予獲選的6組團隊?

    回答:決選時間為111年12月10日(六)，屆時選出6組獲選團隊後，將進行獎金給付作業，由計畫隊長及計畫核銷負責人代表簽領。



    🚩問題:請問有關計畫啟動金的核銷的方式為何?

    回答:計畫啟動金總額為新台幣七萬元，主辦單位將提供相關核銷辦法。計畫啟動金將為兩期進行核銷:第一期款項支付新台幣四萬元，第二期款項支付三萬元。獲獎團隊必須合理採購，檢附相關發票或單據，才能進行核銷。如核銷金額未達各期支付總金額，無法以團隊成員個人名義進行簽領。 上述各階段核銷認可，將由後山黑客松導師檢核計畫及單據
